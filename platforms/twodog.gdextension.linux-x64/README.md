# 2dog.gdextension.linux-x64

Linux x64 native libraries (`libgodot-gdext-{release,debug,editor}.so`) for
[2dog.gdextension](https://www.nuget.org/packages/2dog.gdextension/) - the
GDExtension-based 2dog stack (no mono module, no GodotSharp).

Reference this package alongside `2dog.gdextension` for Linux support.
Variant selection follows `$(TwoDogVariant)` (`release` default, `debug`, `editor`).
