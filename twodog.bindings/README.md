# 2dog.bindings

GodotSharp-compatible C# bindings for Godot Engine over the GDExtension
interface, generated from `extension_api.json` - no mono module, no GodotSharp.

Part of the [2dog.gdextension](https://www.nuget.org/packages/2dog.gdextension/)
stack; usable standalone against any libgodot host.
